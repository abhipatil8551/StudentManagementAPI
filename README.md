# Student Management API

A RESTful Web API built using ASP.NET Core 8, Entity Framework Core, SQL Server, and JWT Authentication.

## Features
- User Registration
- User Login with JWT Authentication
- CRUD Operations for Students
- Swagger API Documentation
- Entity Framework Core
- SQL Server Database
- Repository Pattern
- Exception Handling
- Logging

## Technologies Used
- ASP.NET Core 8
- C#
- Entity Framework Core
- SQL Server
- JWT Authentication
- Swagger

## How to Run
1. Clone the repository
2. Update the connection string in appsettings.json
3. Run:
   ```bash
   dotnet restore
   dotnet ef database update
   dotnet run
   ```
4. Open:
   ```
   https://localhost:5001/swagger
   ```
